# How to Run
### 1. Install Android Studio and Flutter
Search Google
### 2. Clone project
```
git clone https://github.com/lamit1/fb_flutter
```
### 3. Build project:
```
flutter pub get
```
### 4. Run project
#### Connect Physical Device:
- Enable `Debug USD` and `Install via USB` in Setting on physical device if you connect using USB
#### Run:
```
flutter run
```
